contactmap README
-----------------

Description:

	contactmap is a collection of scripts for converting and displaying nucleic acid origami contact map files. A contact map is the most basic specification of a scaffold nucleic acid origami, detailing which staple bases connect to which scaffold bases in the origami design (and vice versa).

Documentation:

	https://contactmap.readthedocs.io

	(The file docs/index.html in this repository gives the same information).
